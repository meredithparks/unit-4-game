Crystal Collectors!

Directions

  • 1. A random number is generated on the page
  • 2. Four gems are also displayed as buttons, each with a different hidden value
  • 3. Each time the player clicks on a button, that hidden value is added to their total score
  • 4. If the player matches their score to the random number, they win the game!
  • 5. When the game is restarted, a new random number and new gem values are assigned
